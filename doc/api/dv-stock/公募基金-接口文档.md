# 公募基金模块 - 接口文档

> **文档来源**：`dv-stock` 模块 `FundController`
> **更新时间**：2026-04-30
> **接口前缀**：`/fund`

## 接口列表总览

| 接口名称 | 请求方式 | 接口路径 | 说明 |
|---|---|---|---|
| 分页查询公募基金基础信息 | POST | `/fund/page/list` | 分页查询基金基础数据 |
| 查询公募基金基础信息列表 | POST | `/fund/list` | 不分页查询基金基础数据 |
| 查询公募基金详情 | GET | `/fund/detail/{fundCode}` | 查询基金基础信息和详情信息 |
| 查询基金资产配置明细 | POST | `/fund/assetRatio/list` | 查询基金各类资产占比 |
| 查询基金持仓(股票/债券) | POST | `/fund/portfolio/list` | 按基金类型查询股票和债券持仓 |
| 分页查询场内基金交易排行 | POST | `/fund/exchangeRank/page/list` | 查询指定交易日的场内基金排行 |
| 分页查询香港基金排行榜 | POST | `/fund/hkRank/page/list` | 查询指定交易日的香港基金排行 |
| 分页查询货币型基金排行榜 | POST | `/fund/moneyRank/page/list` | 查询指定交易日的货币型基金排行 |
| 分页查询理财基金排行榜 | POST | `/fund/financialRank/page/list` | 查询指定交易日的理财基金排行 |

## 通用响应结构

```json
{
  "isSuccess": true,
  "code": 200,
  "message": "success",
  "data": {}
}
```

| 字段名 | 类型 | 说明 |
|---|---|---|
| isSuccess | Boolean | 是否成功 |
| code | Integer | 返回码，成功通常为 `200` |
| message | String | 返回消息 |
| data | Object | 业务数据 |

## 1. 分页查询公募基金基础信息

### 基本信息
- **接口路径**：`POST /fund/page/list`
- **接口说明**：分页查询基金基础信息
- **Content-Type**：`application/json`

### 请求参数

#### Request Body
```json
{
  "pageNum": 1,
  "pageSize": 20,
  "fundCode": "000001",
  "fundShortName": "华夏",
  "fundType": "STOCK",
  "isActive": 1
}
```

| 参数名 | 类型 | 必填 | 说明 |
|---|---|---|---|
| pageNum | Integer | 是 | 页码 |
| pageSize | Integer | 是 | 每页数量 |
| fundCode | String | 否 | 基金代码 |
| fundShortName | String | 否 | 基金简称 |
| fundType | String | 否 | 基金类型编码 |
| isActive | Integer | 否 | 是否有效，`0` 否，`1` 是 |

### 响应数据

#### 成功响应
```json
{
  "isSuccess": true,
  "code": 200,
  "message": "success",
  "data": {
    "total": 1,
    "size": "20",
    "page": "1",
    "list": [
      {
        "id": 1,
        "fundCode": "000001",
        "fundShortName": "华夏成长",
        "fundType": "STOCK",
        "pinyinAbbr": "HZCZ",
        "pinyinFull": "HUAXIACHENGZHANG",
        "isActive": 1,
        "source": "akshare",
        "remark": null
      }
    ]
  }
}
```

#### `data.list` 字段说明
| 字段名 | 类型 | 说明 |
|---|---|---|
| id | Long | 主键 ID |
| fundCode | String | 基金代码 |
| fundShortName | String | 基金简称 |
| fundType | String | 基金类型编码 |
| pinyinAbbr | String | 拼音缩写 |
| pinyinFull | String | 拼音全称 |
| isActive | Integer | 是否有效，`0` 否，`1` 是 |
| source | String | 数据来源 |
| remark | String | 备注 |

### 注意事项
- 该接口返回分页结构 `PageListVo<FundInfoVo>`
- 文档未发现该接口额外默认分页兜底逻辑，建议前端始终显式传 `pageNum`、`pageSize`

## 4. 查询基金资产配置明细

### 基本信息
- **接口路径**：`POST /fund/assetRatio/list`
- **接口说明**：查询基金各类资产在报告期内的占比情况
- **Content-Type**：`application/json`

### 请求参数

#### Request Body
```json
{
  "code": "000001",
  "type": "STOCK",
  "reportDateStart": "2025-01-01",
  "reportDateEnd": "2025-12-31"
}
```

| 参数名 | 类型 | 必填 | 说明 |
|---|---|---|---|
| code | String | 否 | 基金代码 |
| type | String | 否 | 资产类型编码 |
| reportDateStart | LocalDate | 否 | 报告开始日期 |
| reportDateEnd | LocalDate | 否 | 报告结束日期 |

### 响应数据

#### 成功响应
```json
{
  "isSuccess": true,
  "code": 200,
  "message": "success",
  "data": [
    {
      "fundCode": "000001",
      "reportDate": "2025-12-31",
      "assetTypeCode": "stock",
      "assetType": "股票",
      "positionRatio": 87.35,
      "source": "akshare"
    }
  ]
}
```

#### `data` 字段说明
| 字段名 | 类型 | 说明 |
|---|---|---|
| fundCode | String | 基金代码 |
| reportDate | LocalDate | 报告日期 |
| assetTypeCode | String | 资产类型编码 |
| assetType | String | 资产类型名称 |
| positionRatio | BigDecimal | 仓位占比 |
| source | String | 数据来源 |

### 注意事项
- `code` 和 `type` 不能同时为空，否则返回 `INVALID_INFO`
- 报告期建议按季度末或业务实际报告期传入

## 5. 查询基金持仓(股票/债券)

### 基本信息
- **接口路径**：`POST /fund/portfolio/list`
- **接口说明**：根据基金类型查询股票持仓、债券持仓或两者都查
- **Content-Type**：`application/json`

```json
{
  "code": "000001",
  "type": "MIX_BALANCED",
  "reportDateStart": "2025-01-01",
  "reportDateEnd": "2025-12-31"
}

| 参数名 | 类型 | 必填 | 说明 |
|---|---|---|---|
| code | String | 是 | 基金代码 |
| type | String | 否 | 基金类型编码，不传则系统自动识别 |
| reportDateStart | LocalDate | 否 | 报告开始日期 |

#### 成功响应

{
  "isSuccess": true,

      }
    ],
    "bondList": [
      {
        "fundCode": "000001",
        "reportDate": "2025-12-31",
        "reportPeriod": "2025Q4",
        "bondCode": "019742",
        "bondName": "24国债12",
        "navRatio": 5.16,
        "holdingMv10k": 8120.55,
        "rankNo": 1,
        "source": "akshare"
      }
    ]
  }

```

#### `data` 字段说明
| 字段名 | 类型 | 说明 |
|---|---|---|
| fundCode | String | 基金代码 |
| fundType | String | 基金类型编码 |
| queryStock | Boolean | 是否查询股票持仓 |
| queryBond | Boolean | 是否查询债券持仓 |
| stockList | List | 股票持仓列表 |
| bondList | List | 债券持仓列表 |

#### `stockList` 字段说明
| 字段名 | 类型 | 说明 |
|---|---|---|
| fundCode | String | 基金代码 |
| reportDate | LocalDate | 报告日期 |
| reportPeriod | String | 报告期 |
| stockCode | String | 股票代码 |
| stockName | String | 股票名称 |
| navRatio | BigDecimal | 净值占比 |
| holdingShares10k | BigDecimal | 持股数量，单位万股 |
| holdingMv10k | BigDecimal | 持仓市值，单位万元 |
| rankNo | Integer | 排名序号 |
| source | String | 数据来源 |

#### `bondList` 字段说明
| 字段名 | 类型 | 说明 |
|---|---|---|
| fundCode | String | 基金代码 |
| reportDate | LocalDate | 报告日期 |
| reportPeriod | String | 报告期 |
| bondCode | String | 债券代码 |
| bondName | String | 债券名称 |
| navRatio | BigDecimal | 净值占比 |
| holdingMv10k | BigDecimal | 持仓市值，单位万元 |
| rankNo | Integer | 排名序号 |
| source | String | 数据来源 |

### 注意事项
- `code` 不能为空，否则返回 `INVALID_INFO`
- `type` 不传时，系统会先根据基金代码查询基金基础信息，再自动识别基金类型
- `type` 若传入非法枚举值，会返回 `INVALID_INFO`
- `reportDateStart` 晚于 `reportDateEnd` 时，会返回 `INVALID_INFO`
- 若基金类型只支持某一类持仓查询，则另一类列表为空，并通过 `queryStock` / `queryBond` 明确标识

{
